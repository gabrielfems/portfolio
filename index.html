Trabalhos e projetos em equipe

hackathon-nasa-space-apps

Projeto idealizado em equipe durante o NASA Space Apps Challenge 2024. Criamos o conceito do GEOMAG STORM, um simulador em realidade aumentada voltado à educação sobre tempestades solares e seus impactos. A proposta combina visualização 3D, gamificação e conteúdo teórico para promover o aprendizado astronômico de forma imersiva.

gibi-lmp

Projeto pessoal desenvolvido com HTML e CSS, voltado à criação de um site para divulgação e arrecadação de doações destinadas à produção de um gibi que retrata a Lei Maria da Penha e a história real da autora.

screen-match-sem-web

Projeto em desenvolvimento com Java e Spring Boot, que consome dados da API pública OMDb para simular uma plataforma de filmes. O objetivo é praticar o consumo de APIs, manipulação de dados e boas práticas com orientação a objetos.

gerenciador-de-tarefas

Projeto pessoal em Java, executado no terminal, que implementa um gerenciador de tarefas com funcionalidades como adicionar, editar, remover, listar, filtrar e gerar relatórios. Desenvolvido para praticar conceitos de POO, herança, encapsulamento e uso de listas/coleções de dados

site-cafeteria

Projeto acadêmico que simula o site de uma cafeteria voltada para desenvolvedores. Criado com HTML, CSS e JavaScript, com o objetivo de aplicar os fundamentos do front-end e desenvolver um layout responsivo e interativo
